  <h1>בניית עולם תלת־ממדי בעזרת ProBuilder 🧱</h1>
  <p><strong>"יצירת בית תלת־ממדי עם חלוקה לחדרים באמצעות כלי הבנייה של Unity"</strong></p>

  <p>
    בחלק זה של המטלה התמקדנו בבניית עולם תלת־ממדי באמצעות <strong>ProBuilder</strong>, כלי הבנייה המובנה של Unity.
    בעזרת הכלי נבנה בית שלם מאפס, תוך שימוש בצורות בסיסיות ועריכת Mesh, ללא שימוש במודלים חיצוניים.
    הדגש היה על יצירת חללים ברורים, מדרגות וחלוקה פנימית לחדרים שונים.
  </p>

  <ul>
    <li>נעשה שימוש ב־<strong>ProBuilder</strong> ליצירת:
      <ul>
        <li>קירות חיצוניים ופנימיים</li>
        <li>רצפות ותקרות</li>
        <li>מדרגות בין מפלסים</li>
      </ul>
    </li>

    <li>מבנה הבית:
      <ul>
        <li><strong>קומה עליונה</strong> – חולקה לשני אזורים:
          <ul>
            <li>סלון מרכזי</li>
            <li>חדר קטן (חדר שינה / עבודה)</li>
          </ul>
        </li>
        <li><strong>קומה תחתונה</strong> – מרתף סגור, נפרד משאר הבית.</li>
      </ul>
    </li>

    <li>החלוקה לחדרים בוצעה באמצעות <strong>קירות פנימיים</strong> שנבנו כ־Meshes נפרדים ב־ProBuilder.</li>

    <li>לכל חדר נוספו <strong>חפצים בסיסיים</strong> (רהיטים ואלמנטים סביבתיים) שנבנו מצורות פשוטות:
      <ul>
        <li>ספרייה, ספה ושולחן בסלון</li>
        <li>רהיטים בסיסיים בחדר הקטן</li>
        <li>ארגזים ואלמנטים תעשייתיים במרתף</li>
      </ul>
    </li>

    <li>בוצע <strong>איחוד אובייקטים</strong> (Merge) ב־ProBuilder עבור חפצים מורכבים,
        על מנת ליצור Mesh אחד נוח לעבודה ולשמירה על היררכיה מסודרת.</li>

    <li>נעשה שימוש ב־<strong>Materials</strong> שונים כדי להבדיל בין סוגי משטחים:
      <ul>
        <li>קירות (כולל טקסטורת Brick)</li>
        <li>רצפות</li>
        <li>רהיטים ואלמנטים פנימיים</li>
      </ul>
    </li>

    <li>המדרגות נבנו ב־ProBuilder ונוסף מעליהן <strong>Collider רמפה בלתי נראה</strong>
        כדי לאפשר תנועה חלקה של השחקן בין המפלסים.</li>

    <li>העולם נבנה כולו <strong>ללא שימוש בנכסים חיצוניים</strong>,
        תוך הסתמכות על ProBuilder וכלי העריכה של Unity בלבד.</li>
  </ul>
